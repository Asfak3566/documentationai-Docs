## Overview

Ashfaq afr provides a centralized space for your project documentation. You create, organize, and collaborate on docs with ease, supporting Markdown, rich text, and embeds. Teams use it to maintain living documentation that stays up-to-date with project changes.

<Callout kind="info">
Ashfaq afr integrates seamlessly with your Git workflow, version control, and CI/CD pipelines.
</Callout>

## Key Benefits

Ashfaq afr streamlines documentation for project teams by offering intuitive organization, real-time collaboration, and powerful search. You reduce onboarding time for new members and keep knowledge accessible across your organization.

## Quick Start

Get your first documentation space running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com/signup` with your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Space" and name it for your project, like "My Project Docs".
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    Create a Markdown file:
    
````markdown
# Welcome to My Project

This is your first doc.

## Quick Setup

```bash
npm install my-project
```
````
  </Step>
  <Step title="Publish & Share" icon="share-2">
    Hit publish and share the link with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Embed API docs example
const client = new AshfaqAfrClient({ apiKey: 'YOUR_API_KEY' });
await client.docs.create({ title: 'API Guide', content: '# Endpoint\nPOST /v1/docs' });
```
```python
# Python integration
from ashfaq_afr import Client
client = Client(api_key='YOUR_API_KEY')
client.docs.create(title='API Guide', content='# Endpoint\nPOST /v1/docs')
```
</CodeGroup>
